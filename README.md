# GW2.app Blish HUD Module

Access your custom [GW2.app](https://gw2.app) lists in game to work on your
goals. Supports waypoint copying, achievement tracking, timers, story
progress, equipped gear, Trading Post watchers, and more.

Open [gw2.app/blish](https://gw2.app/blish) to get started.
